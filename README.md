# MxSDR — Gerador de Material Comercial (SDR)

Ferramenta interna da **MAXIMOS Soluções Empresariais**. Funciona como uma
"máscara" do prompt comercial: a SDR escolhe a empresa, clica em **Gerar** e
copia o prompt pronto para colar na IA (Claude/ChatGPT) junto com o portfólio e
a ata da reunião. A IA devolve os dois documentos — Material de Apoio (Script
SDR) e Cadência de Contatos.

## Como usar

1. Abra o link de publicação (GitHub Pages) ou o arquivo `index.html`.
2. Entre com o seu nome (tela de perfis).
3. Em **Gerar material**, escolha a empresa e clique no botão verde.
4. **Copiar prompt** → cole na IA → anexe portfólio + ata → envie.

## Características

- Arquivo HTML único, sem instalação, abre em qualquer navegador.
- Cadastro de empresas com indicador de completude (4–7 serviços, 2–5 clientes).
- Histórico de gerações por SDR.
- Modelo do prompt editável (Ajustes) e backup/restauração em `.json`.
- Dados salvos no navegador de cada computador (localStorage). **Exporte backup
  periodicamente** em Ajustes e guarde na rede.
- Autoteste: abra com `?teste=1`.
